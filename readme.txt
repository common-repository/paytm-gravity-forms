== Description ==
This plugin allows you to accept payments using Paytm. After setup configuration with Payment Form. he will redirect to Paytm website to complete his transaction and on completion his payment, paytm will send that user back to your website along with transactions details. This plugin uses server-to-server verification to add an additional security layer for validating transactions. Admin can also see payment status for form entries by navigating to Forms > Entries from the menu in admin.

== Installation ==

* Download Paytm Gravity Forms 	
* Upload this all plugin files in "wp-content/plugins/" directory
* Install and activate the plugin from Wordpress admin panel
* Select "Forms > Settings > Paytm Form" from menu list and update Paytm configuration values provided by Paytm team
* Create a new form then goto page with and Click on "Add Form" , insert "Insert Gravity Form" 
* Click on "Forms > Paytm Form" , create paytm form and just mapped with fields 
* Your Paytm Gravity Forms is now setup. You can now accept payment through Paytm.
